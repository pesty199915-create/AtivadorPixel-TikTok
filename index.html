ℹ️ Como obter meu Pixel ID?

No TikTok Ads Manager vá em: Assets > Event Manager > Gerenciar Pixel. Copie o código que começa com "D" ou o ID numérico. Insira apenas o ID (ex: D7FPCA3C77UAP94ROC70).

📋 Instruções Rápidas

1️⃣ Cole seu Pixel ID no campo acima e clique em "Ativar Pixel".

2️⃣ O script do TikTok será carregado automaticamente no navegador.

3️⃣ Vá na aba "Disparar Eventos" e clique nos botões para enviar eventos padrão.

✅ Os eventos ficam salvos no localStorage (não repetem). Use o botão "Resetar" para limpar e testar novamente.
